# Busca_Sabores
 O objetivo dessa aplicação é pesquisar um código específico de um produto em uma planilha excel.
 
# Como utilizar
 Importe uma planilha com os produtos, adicionais e ou sabores e o código e altere o nome dentro da aplicação, também altere os campos de busca pelo nome atribuido na coluna importada. Então execute o aplicativo e ele irá retornar o valor do código ao lado do nome do produto.
 A biblioteca utilizada é openpyxl, necessário a instalação no terminal *pip install openpyxl*
